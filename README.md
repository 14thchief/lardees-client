# Lardees remote chef - client application
This application is the client-side of an open-source remote restaurant application built with React utilizing the redux workflow.

## Technologies and Frameworks
- SCSS and TailwindCSS will be used in combination.
    - SCSS for page styling 
    - TailwindCSS for fast and easy component styling
- GSAP: animations and smooth transitions
- Material Design
- React-icons: Icons
- SWAL: for sweet alerts
- React-hot-toast: for toast notifications

## How to run
1. Clone this application using terminal `git clone URL`
2. run `npm install --save` to install the  latest version of the application dependencies.
3. run `tailwind init` to initialize tailwindCSS
4. run `npm run dev` to start the development server.
    - Your application will be started automatically.
